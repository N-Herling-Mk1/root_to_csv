CLEAN UP — tunnel / processes

Rule zero: bind loopback only, so even a forgotten server is reachable from the node itself and nowhere else. Then close up anyway.

Never bind 0.0.0.0 on a shared server. If a command or tutorial suggests it, use 127.0.0.1 instead and tunnel.
  • Audit listeners — empty output = clean:
    lsof -iTCP -sTCP:LISTEN -a -u $USER
  • Kill anything left over (e.g. a forgotten http.server or code-server):
    kill <PID>
  • Optional — clear lingering VS Code remote processes:
    pkill -u $USER -f vscode-server
  • Local side — a foreground ssh -L tunnel dies with its terminal. Find stray backgrounded ones:
    ps aux | grep '[s]sh -L'
